
# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Overview
This project focuses on predicting the likelihood of heart disease using various machine learning classification algorithms. The workflow includes data cleaning, exploratory data analysis (EDA), preprocessing, feature scaling, model training, and evaluation.

The best-performing model (K-Nearest Neighbors) was selected based on accuracy.

---

## 📊 Dataset
- The dataset contains medical attributes such as:
  - Age
  - Sex
  - Chest Pain Type
  - Resting Blood Pressure
  - Cholesterol Level
  - Fasting Blood Sugar
  - Resting ECG
  - Maximum Heart Rate
  - Exercise-Induced Angina
  - ST Depression
  - Target (Heart Disease)

---

## 🔍 Project Workflow

### 1. Data Cleaning
- Handled missing values
- Removed duplicates
- Checked data consistency

### 2. Exploratory Data Analysis (EDA)
- Visualized feature distributions
- Identified correlations between variables
- Analyzed patterns related to heart disease

### 3. Data Preprocessing
- Encoding categorical variables
- Feature selection
- Splitting dataset into training and testing sets

### 4. Feature Scaling
- Applied Standard Scaling to normalize numerical features

---

## 🤖 Models Used
The following classification algorithms were implemented and compared:

- Logistic Regression
- Naive Bayes
- K-Nearest Neighbors (KNN)
- Decision Tree
- Support Vector Machine (SVM)

---

## 🏆 Model Selection
- **K-Nearest Neighbors (KNN)** achieved the highest accuracy among all models
- Final model selected based on performance metrics

---

## 📈 Evaluation Metrics
- Accuracy
- F1 Score
